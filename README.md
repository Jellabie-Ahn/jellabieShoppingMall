## 주요기능
> Goods
- 상품 등록 간 이지웍 에디터 사용
- 상품 등록 페이지 내부 글(리뷰) 등록 가능
- 상품 등록시 썸네일 생성 
- 파일 업로드 기능

> User
- 로그인 기능 (아이디 비밀번호 체크 등)
- 로그아웃 적용
- 로그인한 회원은 상품을 카트 보관 및 구입 
- 본인 게시판이 아닐경우 삭제 불가(인터셉터)
- 회원 가입시 이메일 인증 기능
- 패스워드 암호화 구현
- 아이디 중복검사

> Admin
- 상품/회원 관리 기능

## 사용기술
